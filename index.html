About Web Bluetooth

The Web Bluetooth API allows websites to communicate with Bluetooth devices securely. Key points:

  • Only works in secure contexts (HTTPS or localhost)
  • User must explicitly grant permission for each device
  • Supported in Chrome, Edge, Samsung Internet, and WebView on Android
  • Not currently supported in Firefox or Safari
